# Engineering Foundations and APIs

Use these resources to build the service layer behind an AI workflow.

## API and Backend Basics

- [FastAPI tutorial](https://fastapi.tiangolo.com/tutorial/) - Fast path to a working backend API.
- [FastAPI testing guide](https://fastapi.tiangolo.com/tutorial/testing/) - Basic test patterns for API behavior.
- [Microsoft REST API guidelines](https://github.com/microsoft/api-guidelines) - Useful for integration-safe API design.

## Systems and Maintainability

- [System Design Primer](https://github.com/donnemartin/system-design-primer) - General system-design foundation for architecture discussions.
- [The Twelve-Factor App](https://12factor.net/) - Practical principles for production-ready service packaging.
- [Google SRE book: release engineering](https://sre.google/sre-book/release-engineering/) - Release discipline and deployment thinking.
