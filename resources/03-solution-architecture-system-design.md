# Solution Architecture and System Design

Use these resources to choose the right technical approach and defend the architecture to engineering, security, and business stakeholders.

## Architecture Foundations

- [System Design Primer](https://github.com/donnemartin/system-design-primer) - General system-design foundation for architecture discussions.
- [Microsoft AI Agents for Beginners](https://github.com/microsoft/ai-agents-for-beginners) - Broad entry point for agent concepts and patterns.
- [Microsoft Agent Framework](https://learn.microsoft.com/en-us/agent-framework/) - Useful for understanding agent application structure.
- [Anthropic: Building Effective AI Agents](https://www.anthropic.com/research/building-effective-agents) - High-signal guidance on when to use agents, when to keep workflows simple, and how to think about composable patterns.
- [OpenAI: A Practical Guide to Building AI Agents](https://openai.com/business/guides-and-resources/a-practical-guide-to-building-ai-agents/) - Practical guide for agent use cases, orchestration, guardrails, and deployment tradeoffs.
- [AWS enterprise agentic AI architecture](https://docs.aws.amazon.com/prescriptive-guidance/latest/govern-architect-agentic-ai/enterprise-architecture.html) - Enterprise architecture and governance lens.
- [AWS: choosing a generative AI service](https://docs.aws.amazon.com/generative-ai-on-aws-how-to-choose/) - Helps compare service and architecture options.

## Pattern Libraries

- [Awesome agentic patterns](https://github.com/nibzard/awesome-agentic-patterns) - Pattern reference for agent design.
- [Awesome agents](https://github.com/kyrolabs/awesome-agents) - Broad agent ecosystem map.
- [Applied ML patterns](https://github.com/eugeneyan/applied-ml) - Practical patterns for production ML and AI systems.
