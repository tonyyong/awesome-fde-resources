# Deployment, Adoption, and Maintainability

Use these resources to move from prototype to controlled pilot, then to a maintainable system owned by the customer or internal team.

## Production Readiness

- [AWS enterprise AI agents best practices](https://aws.amazon.com/blogs/machine-learning/ai-agents-in-enterprises-best-practices-with-amazon-bedrock-agentcore/) - Enterprise agent best practices.
- [AWS AgentOps with Bedrock AgentCore](https://aws.amazon.com/blogs/machine-learning/agentops-operationalize-agentic-ai-at-scale-with-amazon-bedrock-agentcore/) - Operations for agentic systems.
- [The Twelve-Factor App](https://12factor.net/) - Practical deployment and configuration principles.
- [Google SRE book: release engineering](https://sre.google/sre-book/release-engineering/) - Release process, rollout, and rollback discipline.

## Handoff Questions

- Who owns the system after launch?
- What is the rollback path?
- What is versioned: prompts, eval sets, model settings, retrieval config, tools?
- What logs and metrics prove the system is working?
