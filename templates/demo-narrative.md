# Demo Narrative

## 5-Minute Structure

1. Customer context.
2. Pain in the current workflow.
3. Why this use case is worth solving.
4. What the system does.
5. Architecture and tradeoffs.
6. Evaluation evidence.
7. Security and control boundary.
8. Pilot plan and next steps.
